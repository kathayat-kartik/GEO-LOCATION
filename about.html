How ZoneLock Works

ZoneLock functions through a seamless combination of front-end encryption and location verification:

Encryption Based on Location:

When a user encrypts a message, ZoneLock records the device's latitude and longitude coordinates. These coordinates are mathematically integrated into the encryption key, making the ciphertext unique to that location.

Secure Local Storage:

The encrypted message is automatically downloaded as a .encrypted file, ensuring it remains with the user and never leaves the browser.

Decryption Within the Zone:

When attempting to decrypt, ZoneLock checks the user's current location. If it matches the original location within a set radius (e.g., 30 meters), the decryption key is validated and the message is revealed. If not â€” decryption fails.

Privacy by Design:

All encryption and decryption occur locally in the browser using JavaScript â€” no data is uploaded or stored on any external server.
